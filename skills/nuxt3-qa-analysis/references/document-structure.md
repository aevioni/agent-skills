## 简化报告（快速扫描模式）

当用户只需要快速概览时，使用简化格式：

```markdown
## QA 快速扫描结果

**扫描范围**: {文件数} 个文件

**关键发现**:
- [high] {问题} — {文件}
- [medium] {问题} — {文件}
- ...

**建议优先修复**:
1. {最重要的问题和修复方法}
2. ...
```

## 单维度报告

当用户只关注特定维度时（如"帮我检查 SSR 问题"），只输出该维度的完整分析，其他维度省略。
